# 仓库说明

本仓库意在搭建一个博客，记录总结些日常遇到的技术点、知识点。

[博客地址](http://qiana-wei.github.io)

## 使用

**创建新文章**

```shell
hexo new [layout] <title> -p <categorie/fileName>
```

**发布草稿箱文章**

```shell
hexo publish [layout] <title>
```

**发布文章**

```shell
hexo clean && hexo deploy
```
